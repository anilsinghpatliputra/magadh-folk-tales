# Magadh Folk Tales

A folk tale collection from Magadh, and beyond. Let's explore the rich history together.

## Ops
### Run
```
npm run develop
```

### Update
```
npm run build
aws s3 sync public s3://magadh-folk-tales --delete
```

For major changes to pages, run
```
npm run clean
```
